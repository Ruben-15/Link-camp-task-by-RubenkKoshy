## About the Project
LINK CAMP 2025 is a leadership and networking camp aimed at IEEE student members and aspiring innovators.  
This website acts as a landing page for the event, highlighting:

- Event name, date, and venue  
- Purpose of the camp  
- Key focus areas like leadership, networking, and hands-on activities  
- A clear call-to-action for registration  

The design is kept minimal and professional, focusing on clarity and usability rather than heavy animations.

---

## Key Features
- Clean and modern UI inspired by IEEE design standards  
- Fully responsive layout for mobile, tablet, and desktop  
- Clear navigation with smooth scrolling  
- Countdown timer for the event  
- Structured sections: Home, About, Schedule, Registration  
- Lightweight and fast loading
